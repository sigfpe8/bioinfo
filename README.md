This respository is where I'm gradually creating a library to address "Bioinformatics" issues. I came across this area of software development by reading an old book, "Beginning Perl for Bioinformatics" (James Tisdall, 2001). Perl was quite popular at that time, but since then Python has taken its place and conquered the world. There are lots of materials teaching how to use Python to code Bioinformatics routines, including books, YouTube videos and sites. There's even a popular library devoted to this endeavor.

However, since I'm currently in Zig mode, I decided to try my own code and learn a bit of bioinformatics from the bottom up, instead of just using a known library. In that pursuit I'm being guided by the excellent site ["Rosalind"](https://rosalind.info) (highly recommended!). I'm planning to go through all their problems, adding my solutions as I progress.
